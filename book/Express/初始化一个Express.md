# Express

###  初始化一个Express项目

1.输入以下命令
```
expree -e myexpress
```

2.目录结构说明  
<1>bin,存放启动项目的脚本文件 
<2>node_modules, 存放所有的项目依赖库 
<3>public，静态文件(css,js,img) 
<4>routes，路由文件(MVC中的C,controller) 
<5>views，页面文件(Ejs模板) 
<6>package.json，项目依赖配置及开发者信息 （类似于maven的pom）
<7>app.js，应用核心配置文件
